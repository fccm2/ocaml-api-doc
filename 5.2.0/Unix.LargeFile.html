Module Unix.LargeFile

module LargeFile: sig .. end

File operations on large files. This sub-module provides 64-bit variants of the functions Unix.LargeFile.lseek (for positioning a file descriptor), Unix.LargeFile.truncate and Unix.LargeFile.ftruncate (for changing the size of a file), and Unix.LargeFile.stat, Unix.LargeFile.lstat and Unix.LargeFile.fstat (for obtaining information on files). These alternate functions represent positions and sizes by 64-bit integers (type int64) instead of regular integers (type int), thus allowing operating on files whose sizes are greater than max_int.
